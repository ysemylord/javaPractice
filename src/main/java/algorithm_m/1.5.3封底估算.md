## 封底估算
1. 封底估算是什么  
大O记号是定性的估算，没有真针对具体的数值。 
而封底估算是定量的估算，有具体的数值。
2. 封底估算解决了什么问题  
在有限条件下快速地求解出问题的大概值  
3. 举例  
例子1:波塞多尼奥斯测量地球周长  

   经过测量
   1. 地球7.2度角度对应的弧长为787km
   2. 地球的周长为787km*(360/7.2)=39350km
（其中7.2,和787km都是估算的，而且地球知识近似一个圆）
和现在测量出的40000km相差不大
