## 栈混洗    
## 1.定义
一个栈中的元素进入中间栈，然后中间栈中的元素出栈，得到一个新栈的过程。

## 2.判断一个序列是否是另一个序列栈混洗的结果
### 2.1 规律  
如果原序列中元素顺序有(i,j,k],待判断的序列中元素序列有(k,i,j],则带判断的序列不是混洗的结果
